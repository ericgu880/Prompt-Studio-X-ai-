# PromptStudio 设置功能设计文档

> 目标：参考 Eagle 的本地素材管理体验，把 PromptStudio 的设置中心设计成一个「Prompt 工作流中控台」，覆盖本地库管理、采集、整理、搜索、编辑、模型调用、备份、安全与自动化。

---

## 1. 产品定位

PromptStudio 管理的不是单纯文本，而是一组可复用的生成资产：

```text
Prompt 资产
├─ Prompt 文本
├─ Negative Prompt / System Prompt / User Prompt
├─ 变量模板
├─ 模型与参数
├─ 版本历史
├─ 分类与标签
├─ 参考素材
├─ 生成结果
└─ 使用记录
```

因此设置中心需要解决 6 个问题：

1. 我如何安全地保存和备份 Prompt 库？
2. 我如何快速采集外部 Prompt、图片元数据、网页内容？
3. 我如何自动整理标签、文件夹、智能文件夹？
4. 我如何高效搜索和复用 Prompt？
5. 我如何把 Prompt 发送给模型、API 或本地工作流？
6. 我如何保护隐私、密钥和商业创意？

---

## 2. 设置中心信息架构

建议设置中心采用独立窗口或抽屉式页面，左侧为设置导航，右侧为配置内容。顶部保留「搜索设置」。

```text
设置
├─ 设置首页
├─ 通用
├─ 素材库与存储
├─ 导入与采集
├─ 分类与标签
├─ Prompt 编辑器
├─ 模型与 API
├─ 生成与运行
├─ 搜索与索引
├─ 卡片与预览
├─ 快捷键
├─ 导出、备份与同步
├─ 隐私与安全
├─ 插件与自动化
└─ 高级与诊断
```

建议区分两类设置：

| 类型 | 示例 |
|---|---|
| 应用级设置 | 主题、语言、快捷键、更新、全局采集快捷键 |
| 当前库设置 | 库路径、标签规则、索引、备份、智能文件夹、默认模型 |

## 3. 设置首页

首页作为状态总览，不直接堆叠所有开关。

### 核心模块

| 模块 | 内容 |
|---|---|
| 当前库 | 库名称、路径、Prompt 数量、附件数量、数据库大小 |
| 健康状态 | 索引状态、上次备份、缓存占用、重复项数量 |
| 快捷操作 | 立即备份、重建索引、清理缓存、打开库文件夹 |
| 模型状态 | 默认模型、API 连接、本地模型状态 |
| 隐私状态 | 本地优先、联网功能、应用锁定、日志脱敏 |

## 4. 页面功能清单

### 4.1 通用

| 功能 | 说明 |
|---|---|
| 启动时打开 | 上次使用库 / 指定库 / 空白首页 |
| 默认新建位置 | 当前文件夹 / 待整理 / 指定文件夹 |
| 删除行为 | 移入回收站 / 直接删除 / 删除前确认 |
| 回收站保留 | 7 天 / 30 天 / 手动清理 |
| 默认双击行为 | 打开详情 / 编辑 / 复制 Prompt / 运行测试 |
| 更新设置 | 自动检查更新、更新通道、更新日志 |

### 4.2 素材库与存储

| 功能 | 说明 |
|---|---|
| 当前库路径 | 显示路径、打开文件夹、复制路径 |
| 新建/打开/切换库 | 支持多库工作流 |
| 合并库 | 合并另一个 PromptStudio 库 |
| 迁移库 | 移动到外置硬盘、NAS、云盘同步目录 |
| 存储模式 | 复制附件入库 / 引用外部文件 |
| 缓存管理 | 缩略图、搜索索引、临时文件清理 |
| 数据库维护 | 压缩数据库、修复库、检查缺失附件 |

### 4.3 导入与采集

| 功能 | 说明 |
|---|---|
| 拖拽导入 | Markdown、TXT、JSON、CSV、PNG/JPG 元数据 |
| 剪贴板监听 | 复制 Prompt 后弹出保存卡片 |
| 全局采集快捷键 | 快速保存选中文本或网页内容 |
| 浏览器插件 | 保存网页 Prompt、图片和来源 URL |
| 图片元数据解析 | 读取 SD、ComfyUI、Midjourney 等参数 |
| 自动命名 | 首行标题 / 模型名 / 日期 / AI 自动标题 |
| 自动打标签 | 根据关键词、模型、尺寸、来源添加标签 |
| 重复处理 | 跳过 / 创建副本 / 合并版本 / 手动选择 |

### 4.4 分类与标签

| 功能 | 说明 |
|---|---|
| 标签管理 | 新建、重命名、合并、删除 |
| 标签颜色 | 给角色、场景、风格、模型、状态分色 |
| 标签别名 | realistic / 写实 / real photo 归并为同义词 |
| 自动标签规则 | 包含关键词时自动加标签 |
| 智能文件夹 | 按条件自动聚合 Prompt |
| 文件夹密码 | 对敏感文件夹加锁 |
| 重复检测 | 完全文本、归一化文本、语义相似度检测 |
| 待整理规则 | 无标签、无分类、缺模型、缺标题自动进入待整理 |

### 4.5 Prompt 编辑器

| 功能 | 说明 |
|---|---|
| 默认 Prompt 类型 | 文生图、图生图、LLM、角色设定、工作流 |
| 默认字段 | Prompt、Negative、System、User、模型参数 |
| 变量语法 | `{{角色}}`、`{{风格}}`、`{{镜头}}` |
| 变量面板 | 复制或运行前填写变量 |
| 自动保存 | 间隔保存、失焦保存、手动保存 |
| 版本规则 | 每次编辑生成版本 / 手动保存版本 / 重大修改才保存 |
| 差异对比 | 两个版本之间的文本 diff |
| Token 统计 | 字符数、词数、估算 token 数 |
| Prompt 片段库 | 常用镜头、风格、负面词、参数片段 |

### 4.6 模型与 API

| 功能 | 说明 |
|---|---|
| 模型供应商 | OpenAI、Anthropic、Google、Ollama、LM Studio、ComfyUI 等 |
| API Key 管理 | 添加、编辑、测试连接、删除 |
| Base URL | 支持自定义兼容接口 |
| 代理设置 | HTTP / SOCKS / 系统代理 |
| 默认模型 | 按 Prompt 类型设置默认模型 |
| 文件夹默认模型 | 不同文件夹绑定不同模型 |
| 参数预设 | 尺寸、步数、CFG、Seed、温度、Top-p |
| 本地模型连接 | 检查 Ollama、LM Studio、ComfyUI 状态 |
| 费用提示 | 按模型估算消耗，可关闭 |

### 4.7 生成与运行

| 功能 | 说明 |
|---|---|
| 默认运行方式 | 复制到剪贴板 / 调用 API / 发送本地工作流 |
| 复制格式 | 纯文本、Markdown、JSON、OpenAI Messages、MJ、SD、ComfyUI |
| 变量填写 | 复制前或运行前弹窗 |
| 运行历史 | 保存输入、输出、模型、参数和时间 |
| 结果回存 | 输出自动保存为附件或版本 |
| 批量运行 | 最大并发、失败重试、间隔时间 |
| Seed 策略 | 固定 Seed / 每次随机 / 从 Prompt 读取 |
| 输出命名 | 标题、日期、模型、版本组合命名 |

### 4.8 搜索与索引

| 功能 | 说明 |
|---|---|
| 全文索引 | 标题、Prompt、描述、备注、标签 |

### 4.9 卡片与预览

| 功能 | 说明 |

| 显示字段 | 标题、模型、尺寸、标签、收藏、版本、时间 |

### 4.10 快捷键

| 标签快捷键 | 给常用标签绑定按键 |

### 4.11 导出、备份与同步

| 功能 | 说明 |
|---|---|
| 自动备份 | 每天、每周、退出时备份 |
| 备份位置 | 本地磁盘、外置硬盘、云盘目录 |
| 保留策略 | 最近 N 份或保留 N 天 |
| 恢复备份 | 从备份包恢复 |

### 4.12 隐私与安全

| 功能 | 说明 |
|---|---|

## 5. 推荐 MVP 优先级

### P0：第一版必须做

| 功能 | 原因 |
|---|---|
| 当前库路径、打开库文件夹、手动备份 | 建立本地工具信任感 |
| 模型与 API Key 管理 | Prompt 工具核心使用场景 |
| Prompt 编辑器设置 | 决定 Prompt 维护效率 |
| 复制格式设置 | 最常用复用动作 |
| 标签、文件夹、基础智能文件夹 | 管理规模化后的核心能力 |
| 全文索引、重建索引 | 搜索体验基础 |
| 快捷键自定义 | 效率工具必要能力 |
| API Key 安全、日志脱敏 | 保护用户资产 |

### P1：第二阶段

| 功能 | 原因 |
|---|---|
| 自动导入规则 | 降低整理成本 |
| 重复检测 | Prompt 收藏容易重复 |
| 版本 diff | Prompt 迭代频繁 |
| 导出模板 | 方便分享和迁移 |
| 卡片与详情面板自定义 | 提升浏览体验 |
| 运行历史和结果回存 | 形成 Prompt 到结果闭环 |

### P2：高级功能

| 功能 | 原因 |
|---|---|
| 本地语义搜索 | 大型 Prompt 库搜索体验提升明显 |
| AI 自动命名和自动打标签 | 提升整理效率 |
| MCP / 本地 API | 让 Prompt 库被 AI 助手调用 |
| Git 同步 | 适合高级用户版本管理 |
| 插件系统 | 扩展生态 |

## 6. 交互原则

1. **危险操作必须集中在页面底部**，使用红色弱提醒，不和普通设置混在一起。
2. **所有联网相关功能都必须标注隐私影响**，例如云端 Embedding、云端模型调用、遥测。
3. **库级设置需要显示当前库名称**，避免多库切换后误操作。
4. **设置项要给出推荐默认值**，减少新用户决策压力。
5. **所有高级功能默认折叠**，避免影响第一眼理解。
6. **设置搜索必须支持中文关键词**，例如搜索“备份”“API”“快捷键”“标签”。

## 7. UI 原型说明

本设计配套生成了一个深色主题交互式 HTML 原型：

- 左侧导航包含全部设置页面。
- 顶部支持搜索设置项。
- 每页包含标题、说明、状态卡片、设置区、危险操作区。
- 视觉风格参考当前 PromptStudio 截图：暗色背景、圆角卡片、右侧信息密度、灰白按钮、轻量描边。
